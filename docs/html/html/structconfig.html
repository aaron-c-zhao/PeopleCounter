◆ max_area

uint8_t config::max_area

the max area that a single blob can be.

the max area is for detecting huge blobs which might be composed by two individual blobs merged together. if the blob has an area larger than this value, it will be eroded.

◆ sensitivity

uint8_t config::sensitivity

the sensitivity of the adaptive thresholding

the threshold in the second step of adative thresholding is calculated based on the distribution of the value of the pixels of the blobs. This parameters controls how sensitive the second threshold will be, the higher the value the more sensitive it is.

◆ threshold

int16_t config::threshold

the threshold for binarizing the image

The whole thresholding process is divided into two parts. In the first step, all the pixels that has a larger value(the value is negative, so that means a smaller absolute value), will be rendered black(0), of which this parameter is used for. The good choice for this threshold would be the one just above the background noises.
